Technical & platform

SFMC build

  • Table-based HTML · ~640px body · single column on mobile
  • CSS inlined for production (media queries kept in <head>)
  • Modular Content Builder slots — editors swap content, no code
  • Outlook-safe: VML for buttons & hero background image
  • Retina @2x imagery · alt text on every image
  • Web-safe fonts: Poppins → Arial / Helvetica fallback
  • AMPscript tokens: view-in-browser, unsubscribe, preferences
  • Tap targets ≥44px · body text ≥14–16px

Renders on

  • Apple Mail — macOS & iOS
  • Gmail — web, iOS & Android
  • Outlook — Windows 2016–2021 & Microsoft 365
  • Outlook.com, Outlook for Mac & mobile
  • Yahoo Mail & AOL Mail
  • Samsung Mail (Android)
  • Light & dark mode aware

Adding templates to SFMC

  1. Upload the images to Content Builder; copy the hosted URLs.
  2. Create an email in Content Builder and paste the template HTML.
  3. Replace placeholder image URLs with the hosted Content Builder URLs.
  4. Wrap the editable regions as content blocks / slots.
  5. Set real links + AMPscript (unsubscribe, view-in-browser, profile).
  6. Save as a reusable template.
  7. Send a test and preview across clients (Litmus / Email on Acid).
  8. QA, fix, then activate for sending.
